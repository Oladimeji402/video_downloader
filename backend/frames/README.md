# Frame Templates

Place your frame overlay images here.

## Requirements:
- **Format**: PNG with transparency
- **Naming**: `frame-{name}.png` (e.g., `frame-blue.png`, `frame-gold.png`)
- **Size**: Will be automatically scaled to match video dimensions
- **Important**: The center must be **transparent** so the video shows through

## Example Frame Structure:
```
┌─────────────────────────────────┐
│  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  │  <- Colored/decorated border
│  ▓                           ▓  │
│  ▓    TRANSPARENT CENTER     ▓  │  <- This area is transparent
│  ▓    (video shows here)     ▓  │
│  ▓                           ▓  │
│  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  │
└─────────────────────────────────┘
```

## Tips:
- Use image editors like Photoshop, GIMP, or Figma to create frames
- Common aspect ratios: 9:16 (TikTok/Reels), 1:1 (Square), 16:9 (Landscape)
- Add decorative elements in the corners or edges
- Keep border thickness consistent for best results
